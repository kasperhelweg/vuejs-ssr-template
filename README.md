# README #


The project currently uses node 6.7.0

```
$ npm install -g vue-cli
$ vue init molamil-com/vuejs-ssr-template <my-project>
$ cd <my-project>
$ npm install
$ npm run dev
```

To build for production,

```
$ npm run release
```

will build a dist folder.
